Automatically integrate 'menu cards' from www.tagesmenue.ch inside your wordpress restaurant-website

== Description ==

<h3>This plugin let you automatically integrate the 'daily menu', 'daily menu PDF Link' and 'menu cards' as widget inside your wordpress restaurant-website</h3>
You get also an icon inside your editor, where you can add the 'daily menu' and 'menu cards' as shortlink.
For each 'menu cards' you can set your own icon image, the width and the alt-tag (inside the widget area: Appearance->Widgets)
The flip-book 'menu cards' opens in an iframe over the page.

== Installation ==
You manage all your menu items inside the www.tagesmenue.ch portal.
You need a Tagesmenue Key to be able to connect to tagesmenue, you can register in www.tagesmenue.ch and get one.
Thereafter you can set the key under Settings->Tagesmenue
The items are inserted inside your wordpress-page via webservice (http://www.tagesmenue.ch/WebServiceTagesmenue.asmx)

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
2. Select 'tagesmenue.zip' from your computer
3. Install
4. Activate the plugin on the WordPress Plugin Dashboard
5. Place the 'Tagesmenue', 'Tagesmenue Button', 'Tagesmenue Katalog' Widget into a Widget area through the 'Appearance -> Widgets' menu in WordPress.

= Using FTP =

1. Extract 'tagesmenue.zip' to your computer
2. Upload the 'tagesmenue' directory to your '/wp-content/plugins/' directory
3. Activate the plugin on the WordPress Plugins dashboard
4. Place the 'Tagesmenue', 'Tagesmenue Button', 'Tagesmenue Katalog' Widget into a Widget area through the 'Appearance -> Widgets' menu in WordPress.
